## Issues Encountered
- Had difficulty locating the Extensions panel in Cursor due to the sidebar being hidden
- Initially tried to find the Extensions menu manually but was unsure of its location
- The "Clone Repository" button did not respond initially
- Git was not installed, which prevented cloning from working properly
- Some extension behavior (like login prompts) was not immediately clear

## Solutions
- Explored the menu manually to locate the Extensions panel and eventually found it
- Installed Claude Code and Codex extensions, then restarted the application to ensure they worked properly
- Switched to using Command Palette (Ctrl + Shift + P → Git: Clone) as an alternative method
- Installed Git and restarted Cursor to enable repository cloning
- Looked up guidance online and used ChatGPT selectively to clarify steps when stuck
